# Português

O presente programa implementa um jogo da forca em Assembly. o jogo funciona da seguinte forma:

1) primeiro um jogador deve inserir uma palavra para que os outros jogadores adivinhem. Nessa etapa, também é possível inserir uma dica, essa dica será mostrada para os outros jogadores quando forem adivinhar a palavra;

2) Uma vez que o jogador 1 inseriu a palvra, o jogo da forca inicia. Os jogadores devem tentar adivinhar a palavra sugerindo as letras que a compõe uma de cada vez. A cada jogada a letra chutada é colocada em uma lista de letras usadas.

3) O jogo termina com  a vitória dos outros jogadores caso tenham acertado a palavra ou a derrota. ao final do jogo é possível opitar por jogar novamente.

# English

The present program implements an assembly hangman. the game works as follows:

1) first a player must enter a word for the other players to guess. In this step, it is also possible to insert a tip, this tip will be shown to others players when they are going to guess the word;

2) Once player 1 has entered the word, the hangman game begins. Players must try to guess the word by suggesting the letters that make it up one at a time. Every played the kicked letter is placed in a list of used letters.

3) The game ends with the victory of the other players if they have got the word right or defeat. at the end of the game, it is possible to choose to play again.
